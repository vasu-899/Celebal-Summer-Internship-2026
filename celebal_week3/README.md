# Celebal Technologies Summer Internship 2026
## Week 3 Assignment - Subqueries, CTEs, and Window Functions

### Objective
Analyze Superstore sales data using advanced SQL concepts including Subqueries, CTEs, and Window Functions.

### Dataset
- Superstore Sales Dataset (Kaggle)
- 9994 rows, 21 columns

### Files
- `load_data.ipynb` - Data loading + Visualizations
- `week3_task.sql` - All SQL queries
- `week3_insights.md` - Brief insights
- `top_customers.png` - Top 10 customers chart
- `category_sales.png` - Sales by category chart
- `monthly_trend.png` - Monthly sales trend chart
- `regional_sales.png` - Regional sales chart

### SQL Concepts Used
- Subqueries (above average sales, highest order per customer)
- CTEs (total sales per customer, above average customers)
- Window Functions (RANK, ROW_NUMBER, PARTITION BY)
- Combined JOIN + CTE + Window Function query

### Mini Project: Customer Sales Insights
- Top 5 and Bottom 5 customers identified
- Single order customers analyzed
- Above average sales customers filtered
- Highest order value per customer calculated
